# 📊 Sales CRM Dashboard

A modern, responsive **Sales CRM Dashboard** built using HTML, CSS, and JavaScript.  
This project provides a clean UI for managing contacts, deals, activities, reports, and sales forecasting.

---

## 🚀 Features

### 📌 Dashboard
- Total revenue overview
- Open deals tracking
- Win rate statistics
- Average deal size
- Revenue charts (monthly, quarterly, weekly)
- Pipeline stage visualization
- Recent contacts overview

### 👥 Contacts Management
- View all contacts in a structured list
- Search and filter contacts
- Status tags (Active, Lead, Won, Lost, New)
- Company-based grouping

### 📝 Deals Management
- Complete deal tracking table
- Deal value, stage, probability, and close date
- Progress indicators for each deal
- Status-based categorization

### 🕐 Activities
- Recent activity timeline
- Upcoming tasks scheduler
- Color-coded activity indicators

### 📈 Reports
- Monthly revenue analytics
- Deals distribution by stage
- Key performance indicators (KPIs)
- Export and reporting options (UI ready)

### 🔮 Forecasting
- Revenue prediction charts
- Best case / worst case scenarios
- Pipeline confidence breakdown

### ⚙️ Settings
- User profile settings
- Notification toggles
- Integration options (Slack, Google Calendar)
- Editable form inputs

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling (Modern UI design with variables)
- **JavaScript (ES6)** – Interactivity
- **Chart.js** – Data visualization

---

## 🎨 UI Design

- Clean and minimal SaaS-style dashboard
- Soft color palette with accent blue theme
- Responsive layout (desktop optimized)
- Card-based design system
- Smooth hover and interaction effects
